1. Akses Sistem & Login

1

Buka Aplikasi

Akses alamat web aplikasi melalui browser (Chrome/Firefox/Edge).

2

Halaman Login

Masukkan Email (atau NISN) dan Password anda.

Default role akan diarahkan otomatis ke halaman yang sesuai (Santri atau Ustad).

2. Panduan Santri

A. Melakukan Setoran Hafalan

  1. 1

    Klik menu Hafalan Saya atau tombol microphone di Dashboard.

  2. 2

    Pilih Surah yang akan disetorkan dari daftar dropdown.

  3. 3

    Ada dua cara input:

    • Rekam Langsung: Klik tombol rekam, ucapkan hafalan, lalu stop.
    • Upload File: Pilih file audio (.mp3/.wav, maks 5MB) dari HP/Laptop.
  4. 4

    Klik konfirmasi untuk mengirim. Status akan menjadi Pending.

B. Melihat Hasil Review

Setelah Ustad melakukan review, status akan berubah:

Approved (Lulus)

Hafalan diterima, skor Ditambahkan.

Rejected (Ulang)

Hafalan perlu diulang. Lihat feedback ustad.

3. Panduan Ustad / Admin

A. Review Hafalan Masuk

  1. 1

    Masuk ke menu Review Hafalan. Tabel akan menampilkan submission status Pending.

  2. 2

    Klik tombol Review atau ikon detail pada baris santri.

  3. 3

    Dengarkan audio hafalan pada player yang tersedia.

  4. 4

    Berikan Nilai (Score) dan Komentar/Feedback koreksi tajwid/makhraj.

  5. 5

    Klik Approve (Terima) atau Reject (Tolak minta ulang).

B. Manajemen Data

  • Data Santri: Tambah/Edit/Hapus akun santri, reset password santri.
  • Data Hafalan: Input target surah dan jumlah ayat.
